# Getting started

There is currently two ways of connecting to qurantor api. Ones via [API]() which globally accessable regardless of any language and anohter is via [qurantor](https://example.com) installing npm package if you are using it in a javascript project. 

## API

Check the [API](https://qurantor.vercel.app/api) page.

## Installation

Run the following in your javascript project.

<Tabs items={["npm", "yarn"]}>
    <Tab>
    ```bash
    npm i qurantor
    ```
    </Tab>
    <Tab>
    ```bash
    yarn add qurantor
    ```
    </Tab>
</Tabs>
